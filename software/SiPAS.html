SiPAS-tools

Bioinformatic toolkits of RNA-seq data.


Overview

SiPAS-tools integrates a collection of toolkits facilitating research analysis on RNA-seq data. It is a Java based program, combining efficiency and flexibility to realize high-performance computation of various of functions for the next transcriptomics analysis. ##Apps And User Manual SiPAS-tools is designed to simpliy its usage for the convenience of end users. It has a list of applications (Apps). Each app has a parameter file or some parameters. There are the apps of SiPAS-tools:


Parsing

Parsing samples arrcoding to barcodes.


SampleValidation

Validating the RNA samples using genotype data.


Alignment

Geting bam file using STAR.


Counting

Gene qualitificaiton of samples.


Merging

Geting count table of samples.
